# 🫀 식습관 기반 심혈관 질환 예측 프로젝트

---

## 1. 📌 프로젝트 개요

본 프로젝트는 개인의 **식습관 및 건강 데이터**를 기반으로
심혈관 질환 발생 여부를 예측하는 머신러닝 모델을 구축하고,
질병에 영향을 미치는 주요 식습관 요인을 분석하는 것을 목표로 한다.

---

## 2. 🎯 프로젝트 목표

* 식습관과 심혈관 질환 간의 관계 분석
* 머신러닝 모델을 활용한 질병 예측
* 주요 위험 요인 도출 (Feature Importance)
* 예방 관점에서의 해석 제공

---

### 3.2 주요 변수

#### ✔ 건강 관련 변수

* age: 나이
* sex: 성별
* trestbps: 안정 시 혈압
* chol: 콜레스테롤 수치
* thalach: 최대 심박수

#### ✔ 식습관 관련 변수

* fat_intake: 지방 섭취량
* sodium_intake: 나트륨 섭취량
* sugar_intake: 당 섭취량

## 5. 🧩 Feature Engineering

* 식습관 점수 (가중치 기반)
