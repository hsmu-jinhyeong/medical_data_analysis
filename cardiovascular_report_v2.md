## 1. 프로젝트 개요

### 1.1 배경 및 목적

심혈관 질환은 전 세계 사망 원인 1위로, 조기 예측과 예방이 매우 중요하다.
본 프로젝트는 개인의 건강 수치 및 식습관 데이터를 기반으로 심혈관 질환 발생 여부를 예측하고,
위험 요인에 따른 맞춤형 식이 추천을 제공하는 모델을 구축하는 것을 목표로 한다.

### 1.2 초기 프로젝트 목표

- 머신러닝 모델을 활용한 심혈관 질환 이진 분류 (발생/미발생)
- 개인 건강 정보 입력 시 위험도 수치 및 등급 출력
- 위험 요인 기반 음식 추천 시스템 구현
- Gradio를 활용한 웹 UI 제공

### 2.3 데이터 병합 전략

두 데이터셋은 동일한 환자 정보가 아니므로 직접 row 병합이 불가능하다.
따라서 아래 전략을 사용하였다.

1. CVD 데이터에서 심혈관 질환 여부(0/1)별 식습관 평균값 계산
2. cardio_train의 각 행에 질환 여부에 따른 평균 식습관 값 매핑
3. 현실적인 분포 반영을 위해 표준편차 30% 수준의 노이즈 추가
4. 실제 예측 시에는 사용자가 식습관을 직접 입력하여 활용

## 3. 탐색적 데이터 분석 (EDA)

### 3.1 타겟 변수 분포

- 심혈관 질환 없음: **34,608명 (50.5%)**
- 심혈관 질환 있음: **33,892명 (49.5%)**
- 클래스 불균형이 거의 없어 별도의 오버샘플링 불필요

### 3.2 주요 발견

- **나이:** 질환 있는 그룹의 평균 나이가 유의미하게 높음 (약 55세 vs 45세)
- **혈압:** 수축기 혈압이 높을수록 질환 발생률 급증
- **콜레스테롤:** 등급 3(매우높음) 그룹의 질환 비율이 등급 1 대비 약 2배
- **식습관:** 튀긴음식 섭취가 많을수록, 채소·과일 섭취가 적을수록 질환 비율 증가
- **BMI:** 과체중/비만 그룹에서 질환 비율 높음

### 3.3 상관관계

수축기 혈압(ap_hi), 나이(age_years), 콜레스테롤이 타겟 변수와 가장 높은 양의 상관관계를 보였다.
신체활동(active)은 약한 음의 상관관계를 나타냈다.

## 4. 데이터 전처리

### 4.1 이상치 제거

의학적 기준을 적용하여 불가능한 값을 제거하였다.

| 컬럼 | 제거 기준 |
|---|---|
| ap_hi (수축기 혈압) | 60 미만 또는 250 초과 |
| ap_lo (이완기 혈압) | 40 미만 또는 180 초과 |
| 혈압 관계 | 수축기 ≤ 이완기인 경우 제거 |
| height (키) | 140cm 미만 또는 210cm 초과 |
| weight (체중) | 30kg 미만 또는 200kg 초과 |
| BMI | 10 미만 또는 60 초과 |

- 전처리 전: 70,000행 → 전처리 후: 약 67,000행

### 4.2 파생 변수 생성

| 변수명 | 계산 방법 | 의미 |
|---|---|---|
| age_years | age / 365 | 나이 (세 단위) |
| bmi | weight / (height/100)² | 체질량지수 |
| pulse_pressure | ap_hi - ap_lo | 맥압 |
| hypertension | ap_hi≥140 or ap_lo≥90 | 고혈압 여부 |
| bmi_category | BMI 구간별 0~3 | 저체중/정상/과체중/비만 |
| diabetes | gluc ≥ 2 | 당뇨 가능성 |

### 4.3 데이터 분리

- 학습 / 테스트 = 8 : 2 비율
- `stratify=y` 옵션으로 클래스 비율 유지

## 5. 모델링

### 5.1 사용 모델

XGBoost(eXtreme Gradient Boosting)를 최종 모델로 선택하였다.

**선택 이유:**
- 결측치에 강하고 피처 중요도 해석이 용이
- 대용량 데이터에서 빠른 학습 속도
- 앙상블 기반으로 높은 예측 성능

**주요 하이퍼파라미터:**

```python
XGBClassifier(
    n_estimators  = 200,
    max_depth     = 6,
    learning_rate = 0.1,
    random_state  = 42,
    eval_metric   = 'logloss'
)
```

### 5.2 최종 피처 목록 (20개)

**임상 수치 (16개):** 나이, 성별, 키, 체중, BMI, 수축기혈압, 이완기혈압,
맥압, 고혈압여부, 콜레스테롤, 혈당, 흡연, 음주여부, 신체활동, BMI등급, 당뇨

**식습관 (4개):** 과일섭취, 녹색채소섭취, 튀김섭취, 음주량

## 6. 모델 평가

### 6.1 성능 지표

| 지표 | 질환 없음 | 질환 있음 | 전체 |
|---|---|---|---|
| 정확도 (Accuracy) | - | - | **0.7464** |
| 정밀도 (Precision) | 0.74 | 0.75 | 0.75 |
| 재현율 (Recall) | 0.77 | 0.72 | 0.75 |
| F1 Score | 0.75 | 0.74 | 0.75 |
| AUC-ROC | - | - | **0.8185** |

- 테스트 샘플 수: 질환 없음 6,922명 / 질환 있음 6,778명 (총 13,700명)
- AUC-ROC 0.8185로 심혈관 질환 고위험군 식별 능력 양호

### 6.2 피처 중요도 (상위 5개)

| 순위 | 피처 | 중요도 (약) | 비고 |
|---|---|---|---|
| 1 | 수축기 혈압 (ap_hi) | ~0.35 | 임상 수치 |
| 2 | 고혈압 여부 (hypertension) | ~0.31 | 파생 변수 |
| 3 | 콜레스테롤 (cholesterol) | ~0.08 | 임상 수치 |
| 4 | 나이 (age_years) | ~0.05 | 파생 변수 |
| 5 | 음주량 (Alcohol_Consumption) | ~0.04 | 식습관 (신규 추가) |

- 수축기 혈압과 고혈압 여부가 전체 중요도의 약 66%를 차지
- 식습관 피처 중 음주량이 상위 5위에 진입하여 식습관 데이터 추가의 효과 확인

## 7. 음식 추천 시스템

### 7.1 규칙 기반 추천

예측된 위험 요인에 따라 피해야 할 음식을 추천하는 규칙 기반 시스템을 구현하였다.

| 위험 요인 | 조건 | 피해야 할 음식 예시 |
|---|---|---|
| 고혈압 | ap_hi ≥ 140 또는 ap_lo ≥ 90 | 라면, 가공육, 인스턴트 |
| 고콜레스테롤 | cholesterol ≥ 2 | 튀긴음식, 버터, 크림소스 |
| 고혈당/당뇨 | gluc ≥ 2 또는 diabetes = 1 | 흰쌀밥, 단음료, 과자 |
| 과체중/비만 | BMI ≥ 25 | 패스트푸드, 야식, 디저트 |
| 튀김 과다 | 튀김 섭취 월 15회 이상 | 감자튀김, 도넛, 라면 |
| 채소·과일 부족 | 월 9회 미만 | 가공 주스, 육류 위주 식사 |

### 7.2 Gradio 웹 UI

- 슬라이더 및 라디오 버튼으로 개인 정보 입력
- 위험도 게이지 차트 및 식습관 점수 시각화
- 규칙 기반 음식 추천 텍스트 출력
- `share=True` 옵션으로 외부 공유 가능한 URL 생성

**② 데이터 병합 방식 조정**

### 11.1 결론

- XGBoost 모델이 정확도 74.64%, AUC-ROC 0.8185로 양호한 심혈관 질환 예측 성능을 달성하였다.
- 수축기 혈압(~0.35), 고혈압 여부(~0.31), 콜레스테롤(~0.08)이 가장 중요한 예측 변수로 확인되었다.
- 식습관 데이터를 추가함으로써 음주량이 상위 5위 피처에 진입하는 등 모델의 현실 적용성을 높였다.
- 규칙 기반 음식 추천 시스템을 통해 실용적인 건강 정보를 제공할 수 있었다.

### 11.2 한계점

- 두 데이터셋이 동일 환자가 아니므로 식습관 데이터를 통계적 평균으로 매핑한 한계가 있다.
- 실제 의료 데이터가 아니므로 임상 적용에는 전문가 검토가 필요하다.
- 가족력, 스트레스, 수면 등 심혈관에 영향을 미치는 변수가 포함되지 않았다.

### 11.3 향후 개선 방향

- 하이퍼파라미터 튜닝 (Optuna, GridSearchCV)
- K-Fold 교차 검증 적용
- SHAP을 활용한 예측 결과 해석 강화
- 동일 환자의 임상 수치 + 식습관 데이터 확보 시 정확도 향상 기대
